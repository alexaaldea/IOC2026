Font pairing rationale
Barlow Condensed
Structural hierarchy. Militaristic, compact, authoritative. Used for all human-readable titles, headings, and team communication.
Share Tech Mono
Machine-generated data feel. All sensor outputs, timestamps, IDs, and numeric telemetry use monospace to imply precision and authenticity.
Barlow Light
Human voice layer. Radio transcripts, engineer instructions. Light weight differentiates spoken word from system data.
03 — Iconography
Functional Icons — Outline · 18px
Stroke: 1.5px · Color: Signal Green (#39FF7A) on dark surface. All icons use consistent geometry derived from F1/motorsport semantics. No filled icons on primary surfaces.
Status Indicators — 8px dots & bars
LIVE / OPTIMAL / CONNECTED
HIGH / DEGRADING / CAUTION
CRITICAL / OVERHEAT / FAIL
SYNC / DATA / AR ACTIVE
IDLE / OFFLINE / PENDING
Dot indicators replace text labels where space is minimal (AR overlay, smartwatch). Animated pulse only on LIVE green dot. All other states static.
AR Overlay Icons — Minimal · 14px
AR overlay icons: ultra-thin 1.4px stroke, Cyan tint. Lower visual weight to avoid occluding real-world content. No fill — transparency mandatory on headset glass.

07 — Motion Tokens
Live Pulse
1.4s ease-in-out ∞
LIVE dot indicator. Opacity 1→0.3→1. Signals active data stream.
Alert Flash
0.6s ease 3×
Critical alert banner background. Red opacity flash to draw attention on entry.
Data Update
80ms linear
Numeric readout color flash (cyan→white→cyan) when telemetry value changes.
Panel Slide
200ms ease-out
Side panel open/close on tablet. Translate X from edge. No bounce.
Screen Fade
150ms ease
Tab/surface transitions. Opacity 0→1. No slide on AR (motion sickness).
Bar Fill
400ms ease-out
Throttle/brake/ERS progress bars on initial load. Staggered 50ms delay.
AR headset: disable all translate animations. Use opacity only.
Smartwatch: haptic feedback replaces visual motion for critical events.

10 — Operational Thresholds
Tyre Temperature Logic
mobile map
Optimal 80-115C, warning 115-130C, critical above 130C.
wheel HUD
Cold below 70C, optimal 70-110C, warning 110-125C, critical above 125C.
alert copy
Critical copy must include tire position + measured value + recommended action.
System + Race States
race states
Practice, Qualifying, Race, Safety Car: segmented control drives HUD behavior presets.
sync states
Connected devices must expose signal + battery + explicit connect/disconnect affordance.
pit mode
Emergency mode uses red border pulse, checklist progression, and visible abort path.
11 — Interaction Patterns
Hold To Confirm
Use for destructive pit actions (example: ABORT STOP). Minimum hold duration 1500ms. Requires explicit completion feedback and toast event.
phone · pit wall
Draggable Holo Panels
Panels support pointer drag, pin-to-lock, and hide controls. Motion entry uses blur + scale with 0.5s ease-out for spatial legibility.
MR headset
Critical Alert Acknowledge
Critical watch alerts present large icon pulse, system label, value, and a single acknowledge button. Haptic pattern is mandatory.
smartwatch
Hardware-Mimic Controls
Wheel UI uses tactile visual cues (bevel, inset shadow, active LED ring, short press travel) to emulate physical cockpit controls.
steering wheel
Stateful Banner Routing
Primary alert banner is clickable and routes to relevant deep screen (telemetry/strategy) while triggering haptic confirmation.
phone
